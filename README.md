EvisonNet
=====
### 1. 目录说明<br>
* doc:文档和参考文献
* examples:测试数据
* MiddEval3:middlebury评价脚本
* MiddleburyScripts:读取Middlebury数据集的脚本
### 2. 环境说明<br>
* Windows10 or Ubuntu 18.04
* Python3.6 or 3.7,Anaconda3
* CUDA10+cudnn7.5
* Pytorch
